# ✨ Den Magiske Ædelstens Smedje

Et sjovt og afslappende **3D voxel ædelstens-simulator-spil** lavet i TypeScript.

Generér smukke, unikke ædelsten med forskellige former, farver, renheder og magiske egenskaber. Se dem i 3D og saml din egen samling!

## 🎮 Features

- **14 forskellige ædelstenstyper**
- Procedurally genererede voxel-former
- Interaktiv 3D-visning med **Three.js**
- Forskellige renhedsniveauer (1–4 stjerner + God Tier)
- Specielle **magiske egenskaber** (med glow-effekter)
- Samlingssystem med historik
- Download ædelsten som PNG
- Smuk, moderne UI med Tailwind CSS

## 🚀 Sådan kører du spillet

```bash
# Clone repo
git clone https://github.com/Regnefisken/GemSimulator.git
cd GemSimulator

# Installer afhængigheder
npm install

# Start udviklingsserver
npm run dev
```

Åbn derefter [http://localhost:5173](http://localhost:5173) i din browser.

## 🛠 Tech Stack

- **Frontend**: Vite + TypeScript
- **3D / Voxels**: Three.js
- **Styling**: Tailwind CSS
- **Byggeværktøj**: Vite
